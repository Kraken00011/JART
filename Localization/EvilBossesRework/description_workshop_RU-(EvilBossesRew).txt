Evil Bosses Rework - мод, перерабатывающий Мозг Ктулху и Пожирателя миров (вместе с Бактерией прайм (Bacterium Prime), если у вас включён Exxo Avalon)

Также мод перерабатывает некоторые предметы и мобов, так или иначе связанных с этими боссами или их биомами

Дополнительно, в моде есть немного доп контента для искажения и багрянца
Например:
- Дрели из слитков зла
- Искажённый аналог для гробовщика в виде дьявольской винтовки (мушкет будет продаваться у оружейника)
- Искажённый аналог для гнилой вилки (шар боли теперь является аналогом фрикадельки)
- Багряный аналог для нечестивых стрел (а также Хардмодные улучшения для каждой из стрел)
- Рецепты в мерцании, позволяющие получить предмет другого мирового зла

Примечание:
- На данный момент, в моде есть поддержка только для режима Месть, остальные модовые сложности пока что не поддерживаются.
- Для боссов мировых зол пока что нет эксклюзивного для сложности Мастер оружия
